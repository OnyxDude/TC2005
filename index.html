Preguntas a responder

¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es una red de compus conectadas globalmente, mientras que la World Wide Web es un servicio que usa Internet, nos permite acceder a paginas web

¿Cuáles son las partes de un URL?

Protocolo (https), dominio (google.com), ruta(/servicios), parámetros(&color=azul) y fragmento(#proyectos)

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET - Pide información al servidor HEAD - Pide el encabezado, los metadatos POST - Envía datos nuevos PUT - Actualiza datos existentes PATCH - Modifica solo una parte o seccion DELETE - Elimina un datos

¿Qué método HTTP se debe utilizar al enviar un formulario HTML?

POST, es más seguro porque envia datos sin exponerlos en la URL

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

GET, porque solicita datos del servidor

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto?

Significa que la solicitud fue exitosa y no hubo errores

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta un error 404?

Normalmente si porque un error 404 indica que un recurso no fue encontrado, se puede deber a un enlace roto, una URL mal escrita o puede ser error del usuario si modifico la URL

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta un error 500?

Sí porque un error 500 indica un problema en el servidor que requiere intervención del desarrollador

¿Qué significa que un atributo HTML5 esté depreciado o desaprobado?

significa que sigue funcionando pero ya no se recomienda usarlo porque hay mejores formas de hacer lo mismo, ademas de que podrían dejar de funcionar en futuras versiones. Por ejemplo <center>, para centrar texto, pero ahora es mejor usar CSS con text-align: center

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

HTML5 tiene mas etiquetas semánticas, por lo que puedes tener una estructura mas clara, tiene soporte multimedia nativo, mejoras en formularios y APIs ampliadas para aplicaciones web

¿Qué componentes de estructura y estilo tiene una tabla?

<table>, <tr>, <th>, <td>, <thead>, <tbody> y <tfoot>. Para el estilo, se usa CSS como border, padding, text-align, background-color, width y height

¿Cuáles son los principales controles de una forma HTML5?

<input> (text, password, email, number, radio, checkbox, date, file, submit, reset, range, color, search), <select>, <option>, <button>, y <textarea>, tambien date, range, y color para mayor funcionalidad.

Sobre el ciclo de vida y desarrollo de los sistemas de información:

¿Cuál es el ciclo de vida de los sistemas de información?

Planificación, análisis, diseño, implementación, pruebas, despliegue y mantenimiento

¿Cuál es el ciclo de desarrollo de sistemas de información?

Requisitos, diseño, desarrollo, pruebas, y mantenimiento

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Evitar usarlo, por que si lo utilizas mucho puede ser dificil de leer en una pagina o proyecto grande. Seria mejor usarlo en el ultimo de los casos.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Porque una imagen puede no verse de la forma deseada si la resolucion no es la adecuada o si no se estila correctamente y tambien puede afectar el rendimiento de la página (si es muy pesada)

¿Cuál es tu recomendación al elegir unidades de estilo entre %, px y pt?

Usar % si se busca responsividad, px para tamaños fijos, y es mejor evitar pt porque es mejor usar em o px, y si no es para el tamaño de la tipografia no es recomendado

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Porque al reducir el tamaño del archivo CSS acelera la carga del sitio, supongo que se renderiza mas rapido q la version normal
