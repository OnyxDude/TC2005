Laboratorios

Laboratorio 1

¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es una red de computadoras conectadas globalmente, mientras que la World Wide Web es un servicio que usa Internet, nos permite acceder a páginas web.

¿Cuáles son las partes de un URL?

Protocolo (https), dominio (google.com), ruta(/servicios), parámetros(&color=azul) y fragmento(#proyectos)

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET - Pide información al servidor
HEAD - Pide el encabezado, los metadatos
POST - Envía datos nuevos
PUT - Actualiza datos existentes
PATCH - Modifica solo una parte o sección
DELETE - Elimina un dato

¿Qué método HTTP se debe utilizar al enviar un formulario HTML?

POST, es más seguro porque envía datos sin exponerlos en la URL.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

GET, porque solicita datos del servidor.

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto?

Significa que la solicitud fue exitosa y no hubo errores.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta un error 404?

Normalmente sí, porque un error 404 indica que un recurso no fue encontrado. Se puede deber a un enlace roto, una URL mal escrita o puede ser error del usuario si modificó la URL.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta un error 500?

Sí, porque un error 500 indica un problema en el servidor que requiere intervención del desarrollador.

¿Qué significa que un atributo HTML5 esté depreciado o desaprobado?

Significa que sigue funcionando pero ya no se recomienda usarlo porque hay mejores formas de hacer lo mismo, además de que podrían dejar de funcionar en futuras versiones. Por ejemplo, <center>, para centrar texto, pero ahora es mejor usar CSS con text-align: center.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

HTML5 tiene más etiquetas semánticas, por lo que puedes tener una estructura más clara, tiene soporte multimedia nativo, mejoras en formularios y APIs ampliadas para aplicaciones web.

¿Qué componentes de estructura y estilo tiene una tabla?

<table>, <tr>, <th>, <td>, <thead>, <tbody> y <tfoot>. Para el estilo, se usa CSS como border, padding, text-align, background-color, width y height.

¿Cuáles son los principales controles de una forma HTML5?

<input> (text, password, email, number, radio, checkbox, date, file, submit, reset, range, color, search), <select>, <option>, <button>, y <textarea>, también date, range, y color para mayor funcionalidad.

¿Qué tanto soporte HTML5 tiene el navegador que utilizas?

HTML5TEST is dead. No pude tomar la prueba de mi navegador en el sitio web, pero utilizo Firefox y tiene buen soporte para HTML5.

Sobre el ciclo de vida y desarrollo de los sistemas de información:

¿Cuál es el ciclo de vida de los sistemas de información?

Planificación, análisis, diseño, implementación, pruebas, despliegue y mantenimiento.

¿Cuál es el ciclo de desarrollo de sistemas de información?

Requisitos, diseño, desarrollo, pruebas y mantenimiento.

Laboratorio 3

Como ingeniero de software, ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Evitar usarlo, porque si lo utilizas mucho puede ser difícil de leer en una página o proyecto grande. Sería mejor usarlo en el último de los casos.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Porque una imagen puede no verse de la forma deseada si la resolución no es la adecuada o si no se estila correctamente, y también puede afectar el rendimiento de la página (si es muy pesada).

¿Cuál es tu recomendación al elegir unidades de estilo entre %, px y pt?

Usar % si se busca responsividad, px para tamaños fijos, y es mejor evitar pt porque es mejor usar em o px, y si no es para el tamaño de la tipografía no es recomendado.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Porque al reducir el tamaño del archivo CSS, se acelera la carga del sitio. Supongo que se renderiza más rápido que la versión normal.

Laboratorio 4

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias: Java es un lenguaje de programación compilado, y tiene clases y objetos, mientras que JavaScript es un lenguaje de programación interpretado y en vez de clases u objetos, tiene prototipos que son como todo a la vez. El principal uso de Java es para apps de servidor y de escritorio, mientras que JavaScript se usa para desarrollo web
Semejanzas: Ademas del nombre, ambos lenguajes tienen una sintaxis parecida (segun yo) y basada en C

¿Qué métodos tiene el objeto Date? (Menciona al menos 5)

- getDate(): Devuelve el día
- getMonth(): Devuelve el mes
- getFullYear(): Devuelve el año
- getHours(): Devuelve la hora
- getMinutes(): Devuelve los minutos

¿Qué métodos tienen los arreglos? (Menciona al menos 5)

- push(): Añade elementos al final del arreglo.
- unshift(): Añade elementos al inicio del arreglo.
- pop(): Elimina el último elemento del arreglo.
- shift(): Elimina el primer elemento del arreglo.
- slice(): Devuelve una parte del arreglo.

¿Cómo se declara una variable con alcance local dentro de una función?

Se declara usando const, ya que esta variable solo va a existir dentro de la función y no puede ser accedida fuera de ella

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Puede llevar a problemas de mantenimiento y al momento de debugging, por que cualquier parte del código las puede modificar. Además hace que reutilizar codigo sea dificil y aumenta el riesgo de repetir nombres
